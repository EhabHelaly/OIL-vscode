# OIL
OIL (OSEK Implementation Language) syntax highlighting for vscode
it supports file extensions:
* chx
* tpl
* oil
* tpx
* obx
* stx

## OIL Documentation
you can download OIL documentation [here](http://www.irisa.fr/alf/downloads/puaut/TPNXT/images/oil25.pdf)
